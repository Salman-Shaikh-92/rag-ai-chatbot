#RAG AI Chatbot

An AI-powered chatbot built using Retrieval-Augmented Generation (RAG) to answer questions based on uploaded documents.

This project demonstrates how AI can retrieve information from a knowledge base and generate accurate responses using large language models.

## Project Overview

This chatbot uses the RAG architecture to improve the accuracy of AI responses by retrieving relevant documents before generating an answer.

Instead of relying only on the language model's training data, the system searches a knowledge base and feeds relevant information to the AI.

---

## How the System Works

1. User asks a question
2. The system searches relevant documents
3. Relevant text chunks are retrieved
4. The AI model generates an answer based on retrieved context
